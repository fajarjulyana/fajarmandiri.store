=========================================
   Fajar Mandiri Store - Version 1.0
   Before & After Installation Notes
=========================================

BEFORE INSTALLATION
-------------------
1. Pastikan PC/Laptop memiliki spesifikasi minimum:
   - OS: Windows 10/11 (64-bit)
   - RAM: 2 GB atau lebih
   - Storage: 500 MB tersedia
   - Browser modern (Chrome, Edge, Firefox)
2. Tidak ada aplikasi "Fajar Mandiri Store" di sistem.
3. Database SQLite default belum dibuat.
4. Tidak ada shortcut di Desktop atau Start Menu.
5. Tidak ada service terkait yang berjalan.
