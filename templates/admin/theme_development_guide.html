üìã Introduction

This guide will help you create custom wedding invitation themes using our base template. The template is designed to be highly customizable while maintaining consistency and responsive design.

üöÄ Getting Started

  1. Download the base_template.html file
  2. Copy it to create your custom theme
  3. Rename the file (e.g., romantic_theme.html)
  4. Start customizing the CSS variables

üé® CSS Variables Customization

The base template uses CSS custom properties for easy theming. Here are the main variables you can customize:

Color Variables:
  • --primary-color - Main accent color
  • --secondary-color - Secondary accent color
  • --accent-color - Background accent color
  • --text-dark - Dark text color
  • --text-light - Light text color
Typography Variables:
  • --font-primary - Body font
  • --font-script - Script/decorative font
  • --font-serif - Serif font
Layout Variables:
  • --section-padding - Section spacing
  • --card-radius - Card border radius
  • --button-radius - Button border radius

üí° Best Practices

  • Color Contrast: Ensure good contrast between text and background
  • Performance: Optimize images and minimize custom CSS
  • Accessibility: Test with screen readers and keyboard navigation
  • Cross-browser: Test on multiple browsers
  • Mobile-first: Design for mobile devices first

üß™ Testing Your Theme

  1. Create a test wedding invitation with sample data
  2. Test on different screen sizes
  3. Check all animations and interactions
  4. Verify form functionality
  5. Test background music integration
  6. Test RSVP form submission

üöÄ Theme Deployment

To deploy your custom theme:

  1. Save your theme HTML file in the appropriate directory
  2. Add theme preview image to /static/images/wedding_templates/
  3. Update the database with theme information
  4. Test thoroughly before making it available to users
