AFTER INSTALLATION
------------------
1. Aplikasi "Fajar Mandiri Store" terpasang di:
   C:\Program Files\FajarMandiriStore\
2. Database default (fajarmandiri.db) otomatis disiapkan.
3. Shortcut tersedia di:
   - Desktop
   - Start Menu
4. Aplikasi dapat dijalankan langsung dari shortcut.
5. Server lokal Flask otomatis aktif saat aplikasi dijalankan.
6. Browser default akan membuka alamat:
   http://localhost:5001/  (atau otomatis diarahkan ke https://fajarmandiri.store jika online).
7. Semua fitur (Wedding Invitation, CV Generator, Print Order) siap digunakan.

UNINSTALLATION
--------------
1. Buka Control Panel → Programs → Uninstall a program.
2. Pilih "Fajar Mandiri Store" lalu klik Uninstall.
3. Semua file program akan dihapus, kecuali data user di folder:
   C:\Users\<Username>\AppData\Roaming\FajarMandiriStore
