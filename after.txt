INSTRUKSI INSTALASI / UPGRADE:
1. Backup folder project dan database:
   cp -r /path/to/project /path/to/project_backup
   cp fajarmandiri.db fajarmandiri.db.bak

2. Pastikan folder uploads/ dibuat:
   mkdir -p uploads/music
   mkdir -p uploads/prewedding_photos
   chmod 755 uploads

3. Restart Flask server:
   flask run

4. Test upload file dan buka template wedding, pastikan tidak ada error
